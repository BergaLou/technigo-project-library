# Zen Chi – Digital Yoga Library

This project is a digital library for yoga sessions. The goal was to create an interactive platform where users can explore different practices, filter by difficulty level, sort by duration, or get a random recommendation for their daily flow.

# My Approach

I started by sketching a layout on paper, envisioning a clean and calming yoga/rehab site where users could easily find exactly what they were looking for.

Design: I used CSS Grid and Flexbox to ensure a fully responsive layout that maintains its "Zen" feel across all devices.

Functionality: I focused on clean JavaScript logic, using filter() for the levels and the spread operator [...] with sort() to handle data efficiently without mutating the original array.

# Future Improvements

If I had more time to develop this project further, I would:

Interactive Content: Make the cards clickable to open a detailed view with instructional texts and embedded videos.

Personalization: Build a "Favorites" feature where users can save specific sessions and create their own custom yoga plan.
